Overview

What this package does?

This package provides functions to help obtain useful information from your TypeScript source code.

How does it benefit me?

This can help you automate the generation of a documentation website for any project you're working on. There are a lot of libraries out there that help you create static documentation websites from markdown files:

  • Docusaurus
  • Docsify
  • GitBook
  • VitePress

This package can help you automate the generation of the markdown files that explain the public API of your project.

An Example:

The following example shows the information you can get from a TypeScript function.

Given the following code: 

/**
* Function that sums all the numbers in a list.
*
* @param list - The list of numbers to sum
*
* @returns The sum of all the numbers.
*/
export function sum(...list: number[]): number {
    return list.reduce((acc, curr) => acc + curr, 0);
}

The output information in JSON format will look like: 

[
    {
        "path": "relative-path-from-the-root-of-your-project-where-the-typescript-file-is-located",
        "imports": [],
        "declarations": [
            {
                "kind": "function",
                "name": "sum",
                "signatures": [
                    {
                        "line": 8,
                        "return": {
                            "type": {
                                "text": "number"
                            }
                        },
                        "jsDoc": [
                            {
                                "kind": "description",
                                "value": "Function that sums all the numbers in a list."
                            },
                            {
                                "kind": "param",
                                "value": {
                                    "name": "list",
                                    "description": "The list of numbers to sum"
                                }
                            },
                            {
                                "kind": "returns",
                                "value": "The sum of all the numbers."
                            }
                        ],
                        "parameters": [
                            {
                                "name": "list",
                                "type": {
                                    "text": "number[]"
                                },
                                "rest": true
                            }
                        ]
                    }
                ]
            }
        ],
        "exports": [
            {
                "name": "sum",
                "type": "named"
            }
        ]
    }
]

For more examples, have a look at the playground.
